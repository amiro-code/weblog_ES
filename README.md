# Weblog-ES (MVC)

A simple weblog project implemented with **Node.js** and **Express**, following the **MVC architecture pattern**.

This project is developed as an educational exercise to understand backend fundamentals, MVC separation of concerns, and basic CRUD operations.

---

## Features

- Create new blog posts
- Edit existing posts
- Delete posts
- List all posts
- MVC architecture (Model, View, Controller)
